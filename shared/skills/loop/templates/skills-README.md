# Repo Skills

Task-specific playbooks the agent may use or extend during build iterations.

## How to Use

- Before starting a task, check if an existing skill applies.
- If a skill matches, follow it.
- If no skill matches and creating one would save time or reduce errors in future iterations, create a new one.
- Skills live in `skills/<name>/SKILL.md`.

## Creating a Skill

1. Create `skills/<name>/SKILL.md` with:
   - When to use this skill
   - Step-by-step workflow
   - Examples (optional)
   - Supporting files (optional, in the same directory)
2. Update this README to list the new skill.
